Initial setup

Google setup

This setup requires the use of goole apis through oauth. To get started, go to https://console.cloud.google.com/apis/dashboard and create a new project. Once created, go to the credentials tab and create a new oauth client id. Select web application as the application type and add the following to the authorized redirect URIs: "BACKEND_URL/setup/oauth" Example: "https://localhost:3443/setup/oauth" (good for testing, but not for production) Example: "https://phrydge.engphys.ca/setup/oauth" (good for production) Once created, copy the client id and client secret into the form below.
By clicking continue, you will be redirected to a google oauth page. This will be used to authorize the backend for read, modify, and send email permissions on your club gmail account. The authorized permissions will only be used to send notification emails to users, site reports to admins, and monitor for incoming e-transfer auto deposits.
